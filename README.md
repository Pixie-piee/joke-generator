# Joke Generator

A simple Express.js web app that fetches random jokes from the JokeAPI and displays them using EJS.

## Features

- Random joke generation
- Express.js backend
- Axios for API requests
- EJS templating

## Tech Stack

- Node.js
- Express.js
- Axios
- EJS

## Installation

```bash
npm install
npm start
```

Open:

```
http://localhost:3000
```
